Microsoft Excel sheet

A demonstration of MS Excel sheet features

Drop any .xlsx file to the grid to open and edit it here! Or open any .xlsx file from your disk by click to icon on toolbar.
The xlsx files can be opened, edited and saved (by icon) directly in browser, without any communication with server!
Basic MS Excel sheet look, cursor, navigation, editing, selecting, copying, moving, filling, resizing, pasting and cell span.
Configuring key and mouse action schema, dynamic cell styles and borders, outside editing, auto generating pages.
